===============================================================================

Problem: "Permission denied: ./run_all.sh"
Solution: Make it executable:
    chmod +x run_all.sh
    ./run_all.sh

Problem: "Virtual environment not found"
Solution: Create it:
    python3.11 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ./run_all.sh

Problem: "No module named 'X'"
Solution: Install dependencies:
    source venv/bin/activate
    pip install -r requirements.txt
    ./run_all.sh

Problem: "Components not communicating"
Solution: Stop everything and restart:
    pkill -f "charger_module.py|csms.py|cp.py|plot_current.py"
    Wait 3 seconds
    ./run_all.sh

Problem: "Graph window not showing"
Solution: 
    - Check if it's hidden behind other windows
    - Wait 15-20 seconds for data to accumulate
    - Make sure charger_module.py is running
